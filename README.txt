--------------------------------------------------------------------------------
 ABOUT
--------------------------------------------------------------------------------

This repository contains the source code of 'Balsa', a fast, light-weight
implementation of the Random Forest classificiation algorithm. Balsa is
optimised for low memory footprint and CPU usage during classification.

--------------------------------------------------------------------------------
 DOCUMENTATION
--------------------------------------------------------------------------------

The Documentation directory contains a user manual with build instructions,
installation instructions, usage instructions, optimization tips and a short
overview of the theory and jargon. It is recommended to use a Markdown viewer to
read the documentation.
